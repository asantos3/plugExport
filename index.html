This application requires access to your YouTube account. Please authorize to continue.

How To! ▼
  1. Drag plugPlaylist javascript bookmarklet to your bookmark bar.
  2. Navigate to plug.dj, log in and select the playlist that you wish to upload to Youtube (like this). Click on the bookmarklet and a list of Youtube IDs will be downloaded as a .txt file.
  3. Authorize access (link above) to your youtube account - this is a simple open source implementation of YouTube Data API, you can check out the code here if you don't trust the author.
  4. Type a title for your playlist in the small box.
  5. Paste the list of IDs from the .txt file into the large box.
  6. Click on "Add to current playlist" and wait for the task to be completed.
